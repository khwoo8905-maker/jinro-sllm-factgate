# 온프렘 진로상담 sLLM — 모든 추천을 근거에 묶다

*담백 · 2026년 6월 · Qwen3.5-4B 기반 온프렘 / 울산 학생 진로상담*

학생의 정보를 학교 밖으로 내보내지 않으면서 진로상담을 돕는 길은, 학교 안에서 동작하는 소형 언어모델(sLLM)입니다. 방향은 분명합니다. 남는 과제는 신뢰성입니다 — 모델이 제시하는 모든 학과·진로 추천이 지어낸 것이 아니라 실제 근거 자료(커리어넷)에 묶여 있어야 합니다.

근거 없이 그럴듯한 답을 만들어 내는 일은 특정 모델의 결함이 아니라, 대형 모델을 포함한 모든 언어모델이 공통으로 안고 있는 과제입니다. 이 문서는 그 과제를 진로상담이라는 영역에서 어떻게 다루었는지를, 성과와 함께 한계와 시행착오까지 정리한 기록입니다.

> **먼저 밝히는 한계.** 아래 수치(근거 없는 추천 0%, 유효 답변 100%)는 합성 평가셋 50건에 대한 결과이며, 실제 학생 데이터로는 아직 검증하지 못했습니다. 또한 문서에 실재하는 학과를 학생에게 부적합하게 추천하는 경우나 설명 문장의 사실 오류는 측정 범위에 포함하지 못했습니다. 이 기록의 핵심은 특정 수치가 아니라, 추천을 근거에 묶어 신뢰성을 확보한 방법과 그 검증 과정입니다.

## 구성

Qwen3.5-4B를 커리어넷 자료 검색(RAG) 및 도메인 적응과 결합해 온프렘 환경(GB10)에서 동작시켰습니다. 평가는 별도의 모델(qwen3.6-35b)을 심사자로 두어 진행했습니다.

## 신뢰성을 확보한 과정

해법에 이르기까지 두 가지 가설이 빗나갔습니다. 그 과정을 그대로 남깁니다.

먼저 평가 자체를 점검했습니다. 초기 측정에서 근거 없는 추천 비율이 높게 나왔으나, 이는 생성 길이 제한으로 답변이 중간에 잘린 데서 비롯된 착시였습니다. 측정 조건을 바로잡자 수치가 안정되었습니다.

다음으로 학습 데이터를 정제해 다시 학습해 보았습니다. 그러나 결과는 의미 있게 달라지지 않았습니다. 이어 검색 범위가 원인이라고 보고 살폈으나, 추천 학과가 검색 문서에 포함된 경우와 그렇지 않은 경우의 차이가 뚜렷하지 않았습니다.

핵심은 추론 시점에 있었습니다. 모델이 제시한 추천 학과가 실제 근거 문서에 있는지 확인하고, 근거가 없으면 그 추천을 결과에서 제외하는 검증 단계를 두는 방식입니다. 이때 학생의 희망이나 본문 설명처럼 추천이 아닌 부분은 건드리지 않고, 추천 항목만 근거와 대조하는 것이 중요했습니다. 추천과 단순 언급을 구분하지 못하면 멀쩡한 답까지 함께 사라지기 때문입니다.

## 결과 (합성 평가 50건)

- 모델이 제시한 추천의 약 98%는 검증 이전에 이미 근거 자료에 부합했습니다.
- 검증 단계를 거친 뒤 근거 없는 추천은 0%, 유효한 답변은 100%였습니다.
- 교차 검증에서, 답변을 관련 문서와 대조하면 근거 있는 추천이 유지되고 무관한 문서와 대조하면 모두 걸러져, 검증이 실제로 작동함을 확인했습니다.

확신할 근거가 없을 때는 답을 지어내지 않고 "진로교사와의 상담을 권한다"는 안내로 대신합니다. 잘못된 추천보다 정직한 유보가 더 안전하다는 판단입니다.

## 남은 한계

- 합성 평가 50건을 기준으로 하며, 실제 학생 데이터 검증은 아직입니다.
- 문서에 실재하는 학과를 부적합하게 추천하는 경우와, 설명 문장 자체의 사실 오류는 측정하지 못했습니다.

## 정리

신뢰성은 더 큰 모델이 아니라 추론 단계의 설계, 곧 근거 검색과 추천 검증에서 확보했습니다. 작은 모델과 도메인 자료, 추천 검증을 결합하면 데이터를 외부로 내보내지 않고도 현장에서 쓸 수 있는 진로상담 보조가 가능하다는 것이 이 기록의 결론입니다.
